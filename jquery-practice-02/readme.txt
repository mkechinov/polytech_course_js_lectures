0. Продемонстрировать jQuery UI базовые вещи (панели, кнопки, диалоги) и объяснить, что есть JS, а есть CSS (темы).
1. Создать заготовку из HTML, JS и стилей. Сюда же подключить jQuery UI: https://code.jquery.com/ui/1.11.4/jquery-ui.js и CSS https://code.jquery.com/ui/1.11.4/themes/black-tie/jquery-ui.css
2. Сделать функцию, которая рисует на странице N квадратов разного цвета. Цвета выбираются случайным образом из трех: красный, синий, зеленый.
3. Сделать три кнопки, каждая из которых включает/выключает блоки своего цвета.
4. Сделать числовое поле, которое при вводе в него числового значения рисует столько блоков, сколько введено в поле. Если меньше 1, то 50. Если больше 500, то 500.
